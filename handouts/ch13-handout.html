Chapter 13 in Class: Logistic Regression

1. Introduction to Logistic Regression

Logistic regression models a binary outcome (e.g., success/failure, yes/no) using the logistic function, which ensures probabilities are between 0 and 1.

Why Not Use Linear Regression?

Linear regression can predict probabilities outside the valid range (01), which is meaningless for binary outcomes. Logistic regression uses a logit transformation to model probabilities.

Real-Life Examples

  1. Predicting whether a patient has a disease (1 = Yes, 0 = No).
  2. Predicting whether an email is spam (1 = Spam, 0 = Not Spam).

2. Understanding Probabilities, Odds, and Log Odds

Probability

The likelihood of success: \[ P(Y=1) = \frac{\text{Number of Successes}}{\text{Total Observations}} \]

Odds

The ratio of success to failure: \[ \text{Odds} = \frac{P}{1-P} \]

Log Odds

The natural log of odds: \[ \text{Log Odds} = \ln(\text{Odds}) \]

3. The Logistic Function and Why We Exponentiate

The logistic function transforms log odds into probabilities: \[ P(Y=1) = \frac{e^{\beta_0 + \beta_1X}}{1 + e^{\beta_0 + \beta_1X}} \]

Why Exponentiate Coefficients?

Logistic regression outputs coefficients in log odds, which are difficult to interpret. Exponentiating them converts log odds into odds ratios: \[ \text{Odds Ratio} = e^{\text{Log Odds}} \]

Example

If the coefficient for GPA is 1.5: \[ e^{1.5} \approx 4.48 \] This means a 1-unit increase in GPA multiplies the odds of success by 4.48.

4. Step-by-Step Guide to Running Logistic Regression (with broom)

Why Use the broom Package?

The broom package simplifies the process of extracting, tidying, and visualizing regression output, making it easier to interpret results.

Step 1: Simulate Data

# Simulate Data
set.seed(123)
gpa <- runif(200, 2.5, 4.0)  # Predictor variable
admission <- rbinom(200, 1, prob = plogis(-2 + 0.8 * gpa))  # Moderate effect of GPA
dataset <- data.frame(admission, gpa)

Step 2: Fit the Logistic Regression Model

model <- glm(admission ~ gpa, data = dataset, family = "binomial")

Step 3: Use broom::tidy() for Clean Output

tidy_model <- tidy(model, exponentiate = TRUE, conf.int = TRUE)
tidy_model
## # A tibble: 2  7
##   term        estimate std.error statistic  p.value conf.low conf.high
##   <chr>          <dbl>     <dbl>     <dbl>    <dbl>    <dbl>     <dbl>
## 1 (Intercept)   0.0251     1.27      -2.91 0.00365   0.00196     0.288
## 2 gpa           3.88       0.395      3.43 0.000595  1.82        8.63

What Does tidy(model, exponentiate = TRUE, conf.int = TRUE) Do?

  1. tidy():

    • Extracts the models coefficients and organizes them into a tidy data frame.

    • Includes statistical estimates, standard errors, test statistics, and p-values.

  2. exponentiate = TRUE:

    • Converts the coefficients from log odds to odds ratios by applying the exponential function:

    \[\text{Odds Ratio} = e^{\text{Log Odds}}\]

  3. conf.int = TRUE:

    • Adds confidence intervals for the coefficients and exponentiates them to reflect odds ratios.

Interpreting the Intercept

  • The Intercept represents the odds of the outcome (e.g., admission = 1) when the predictor variable (e.g., GPA) is at 0.

    • Estimate: Odds Ratio=0.025:

      • The odds of admission are very small when GPA = 0.

    • Confidence Interval (0.002 to 0.29):

      • With 95% confidence, the odds of admission with 0 GPA range between 0.002 and 0.29.

    • p-value (<0.001):

      • The intercept is statistically significant, suggesting non-random odds when GPA = 0.

Interpreting the GPA Coefficient

  • The GPA coefficient tells us how the odds of the outcome change for a 1-unit increase in GPA.

    • Estimate: Odd Ratio=3.88:

      • For every 1-unit increase in GPA, the odds of admission increase by a factor of 3.88.

    • Confidence Interval (1.82 to 8.63):

      • With 95% confidence, the odds of admission for a 1-unit increase in GPA are between 1.82 and 8.63 times the baseline.

    • p-value (<0.001):

      • The coefficient is statistically significant, meaning GPA is a strong predictor of admission.

How to Write These Results in Plain English

  1. Intercept:

    • When GPA is 0, the odds of admission are very low (odds ratio = 0.03).

    • With 95% confidence, the odds of admission at GPA = 0 are between 0.002 and 0.29.

  2. GPA:

    • For every 1-point increase in GPA, the odds of admission increase by 3.88 times.

    • With 95% confidence, the odds of admission increase between 1.82 and 8.63 times for each additional point in GPA.

    • GPA is a statistically significant predictor of admission (p < 0.001).

Confidence Intervals and Statistical Significance

  • Confidence intervals that exclude 1 suggest that the predictor has a significant effect:

    • If \(1 \notin [\text{Conf.Low}, \text{Conf.High}]\), the predictor is significant.

    • Example: GPAs confidence interval (1.64, 3.00) excludes 1, so GPA is statistically significant.

  • Confidence intervals that include 1 indicate a non-significant effect:

    • If \(1 \in [\text{Conf.Low}, \text{Conf.High}]\), the predictor might not be meaningful.

Odds Ratios vs.Probability

  • Odds ratios tell us how much the odds of an outcome increase (or decrease) for a 1-unit change in the predictor.

  • To calculate probabilities, you can use the logistic regression equation:

\[ P(Y = 1) = \frac{e^{\beta_0 + \beta_1X}}{1 + e^{\beta_0 + \beta_1X}} \]

Key Points to Remember

  1. Exponentiating Coefficients:

    • Transforms log odds into odds ratios, which are easier to interpret.

  2. Odds Ratio Interpretation:

  • \(\text{Odds Ratio} > 1\): Increases odds of the outcome.
  • \(\text{Odds Ratio} < 1\): Decreases odds of the outcome.
  • \(\text{Odds Ratio} = 1\): No effect on the odds.

  1. Confidence Intervals:

    • Provide a range of plausible values for the odds ratio.

  2. Statistical Significance:

    • Determined by the ppp-value and whether the confidence interval excludes 1.

Step 4: Predict Probabilities

predict(model, newdata = data.frame(gpa = 3.5), type = "response")
##         1 
## 0.7431673

This calculates the probability of admission for a GPA of 3.5.

5. Visualizing Logistic Regression

DOT Plot for Odds Ratios

# Extract Odds Ratios and Confidence Intervals, Exclude Intercept
tidy_plot <- tidy(model, conf.int = TRUE, exponentiate = TRUE) %>%
  filter(term != "(Intercept)")  # Exclude the intercept

# Create DOT Plot
ggplot(tidy_plot, aes(x = estimate, y = term)) +
  geom_point(size = 4, color = "blue") +  # Odds ratio point estimates
  geom_errorbar(aes(xmin = conf.low, xmax = conf.high), width = 0.2, color = "black") +  # Confidence intervals
  geom_vline(xintercept = 1, linetype = "dashed", color = "red", size = 0.8) +  # Reference line at OR = 1
  labs(title = "Dot Plot of Odds Ratio with Confidence Interval",
       x = "Odds Ratio (95% CI)",
       y = "Predictor Variables") +
  theme_apa()

Bar Graph of Group Means

# Create Horizontal Bar Graph
ggplot(tidy_plot, aes(y = term, x = estimate)) +
  geom_bar(stat = "identity", fill = "skyblue", width = 0.5) +  # Horizontal bars for odds ratios
  geom_errorbarh(aes(xmin = conf.low, xmax = conf.high), height = 0.2, color = "black") +  # Horizontal confidence intervals
  geom_vline(xintercept = 1, linetype = "dashed", color = "red", size = 0.8) +  # Reference line at OR = 1
  scale_x_log10(breaks = c(0.5, 1, 2, 3, 4)) +  # Logarithmic scale for odds ratios
  labs(title = "Horizontal Bar Graph of Odds Ratios with Confidence Intervals",
       x = "Odds Ratio (95% CI)",
       y = "Predictor Variables") +
  theme_apa()

6. Evaluating Model Fit with pROC and ROC Curve

What is an ROC Curve?

  • Receiver Operating Characteristic (ROC) Curve:
    • A graphical representation of a classifiers performance at various threshold levels.
    • Plots the True Positive Rate (TPR) (Sensitivity) against the False Positive Rate (FPR) (1 - Specificity).
  • Area Under the Curve (AUC):
    • A single scalar value representing the overall performance of the classifier.
    • Ranges between 0 and 1:
      • AUC = 0.5: Model performs no better than random chance.
      • AUC = 1: Perfect classifier.

Fit the Logistic Regression Model

Calculate the ROC Curve and AUC

# Load pROC Library
library(pROC)

# Predict Probabilities
predicted_probs <- predict(model, type = "response")

# Generate ROC Curve
roc_curve <- roc(dataset$admission, predicted_probs)

# Display AUC
auc_value <- auc(roc_curve)
print(paste("AUC:", round(auc_value, 3)))
## [1] "AUC: 0.65"

Plot the ROC Curve

# Plot the ROC Curve
plot(roc_curve, col = "blue", lwd = 2, main = "ROC Curve for Logistic Regression")

Enhanced ROC Curve Visualization with ggplot2

Why Use the ROC Curve?

  1. Threshold Independence:
    • The ROC curve evaluates performance across all possible classification thresholds.
  2. Robust Comparison:
    • The AUC allows you to compare the performance of different logistic regression models.
