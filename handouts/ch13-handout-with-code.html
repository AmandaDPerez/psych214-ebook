Introduction to Interactions in Linear Models

What is an Interaction?

An interaction occurs when the effect of one predictor variable on an outcome variable changes depending on the level of another predictor variable. Interactions are common in the real world, where combined factors often influence outcomes differently than when considered alone.

  • Example: In a study of exercise and stress levels on mood, exercise might improve mood more for people with high stress than for those with low stress. Here, stress level moderates the effect of exercise on mood.

Main Effects vs. Interaction Effects

  • Main Effect: The main effect of a predictor shows its average influence on the outcome, ignoring other predictors.
    • Example: The main effect of diet on energy levels would show how diet affects energy on average, without considering other factors like sleep quality.
  • Interaction Effect: An interaction effect occurs when the effect of one predictor depends on the level of another predictor.
    • Example: If diet quality has a stronger effect on energy when sleep quality is high than when it is low, there is an interaction effect between diet and sleep.

Using Means to Understand Interactions

  • Overall Mean: Represents the average outcome across all predictor combinations.

  • Group Means: Group means are the averages for specific predictor combinations (e.g., diet and sleep levels).

    Example: If we examine diet (high vs. low) and sleep (high vs. low) on energy:

    • Group means would be calculated separately for high-diet/high-sleep, low-diet/low-sleep, etc.
    • If the effect of diet on energy varies by sleep quality (e.g., stronger for high sleep), an interaction is present.

    By comparing group means, we can detect interactions:

    • Cross-over interaction: Group means show opposite effects (e.g., diet improves energy under high sleep but reduces it under low sleep).
    • Magnitude interaction: Group means show different effect sizes without changing direction (e.g., diet improves energy more under high sleep but still improves under low sleep).

Types of Interactions: Cross-Over vs. Magnitude

  1. Cross-Over Interaction: Occurs when the effect of one predictor reverses at different levels of another predictor.
    • Example: Suppose therapy reduces stress for adults but increases stress for teens due to peer pressure.
  2. Magnitude Interaction: Occurs when the effect of one predictor is stronger or weaker at different levels of another predictor, but the direction remains the same.
    • Example: Exercise improves mood more for high-stress individuals than low-stress ones, but it improves mood in both cases.

Setting Up an Interaction in R

To set up an interaction term in a linear model in R, we use the * operator. Here’s an example with simulated data for sleep quality and stress level predicting productivity. We use mutate() from dplyr to apply effect coding (-1 and 1) to categorical variables.

# Set up a controlled dataset with specified means for each group
set.seed(123)

# Create data for each combination with clear mean differences
high_sleep_high_stress <- tibble(
  sleep_quality = factor(rep("High", 25)),
  stress_level = factor(rep("High", 25)),
  productivity = rnorm(25, mean = 90, sd = 3)
)

high_sleep_low_stress <- tibble(
  sleep_quality = factor(rep("High", 25)),
  stress_level = factor(rep("Low", 25)),
  productivity = rnorm(25, mean = 50, sd = 3)
)

low_sleep_high_stress <- tibble(
  sleep_quality = factor(rep("Low", 25)),
  stress_level = factor(rep("High", 25)),
  productivity = rnorm(25, mean = 55, sd = 3)
)

low_sleep_low_stress <- tibble(
  sleep_quality = factor(rep("Low", 25)),
  stress_level = factor(rep("Low", 25)),
  productivity = rnorm(25, mean = 85, sd = 3)
)

# Combine all groups into one dataset
dataset <- bind_rows(high_sleep_high_stress, high_sleep_low_stress,
                     low_sleep_high_stress, low_sleep_low_stress) %>%
  mutate(
    sleep_code = if_else(sleep_quality == "High", 1, -1),
    stress_code = if_else(stress_level == "High", 1, -1),
    interaction = sleep_code * stress_code
  )

# Example code to calculate means and create an APA-styled table with Times New Roman font
dataset %>%
  group_by(sleep_quality, stress_level) %>%
  summarise(mean_productivity = mean(productivity), .groups = "drop") %>%
  # Rename headers: capitalize and replace underscores with spaces
  rename("Sleep Quality" = sleep_quality,
         "Stress Level" = stress_level,
         "Mean Productivity" = mean_productivity) %>%
  kable("html", align = "c", 
        table.attr = "style='font-family: Times New Roman; font-size: 12px; width: 60%; margin-left: auto; margin-right: auto;'") %>% # Apply inline CSS
  kable_styling(full_width = FALSE, position = "center") %>%
  column_spec(3, bold = TRUE) %>%               # Make numeric columns bold
  row_spec(0, bold = TRUE, underline = TRUE)    # Header row in APA style

Interpreting the Output

  • Intercept: The baseline productivity for the reference levels (e.g., low sleep, low stress).
  • Main Effects: Each main effect shows how one category level differs from the average across both levels.
  • Interaction Term: A significant term indicates that the impact of sleep on productivity changes depending on stress level.

Visualizing Interactions with Interaction Plots

Visualizing interactions helps reveal patterns. Here’s how to use ggplot2 to create an interaction plot for the sleep quality and stress level example, with error bars.

# Plot interaction with error bars
ggplot(dataset, aes(x = sleep_quality, y = productivity, fill = stress_level)) +
  stat_summary(fun = mean, geom = "bar", position = "dodge") +
  stat_summary(fun.data = mean_cl_normal, geom = "errorbar", position = position_dodge(0.9), width = 0.25) +
  labs(title = "Interaction between Sleep Quality and Stress on Productivity", x = "Sleep Quality",
       y = "Productivity") +
  scale_fill_manual(values = color_blind_palette) +
  theme_apa()

  • Interpreting the Plot:
    • Cross-over interactions show lines crossing, meaning one predictor reverses the effect of the other.
    • Magnitude interactions show different slopes without crossing, indicating the effect size changes but the direction remains the same.

APA Reporting for Main Effects and Interaction

Results

A multiple regression analysis was conducted to examine the effects of Study Hours and Sleep Hours on Exam Scores and to test for an interaction between Study Hours and Sleep Hours. Descriptive statistics for each variable are presented in Table 1.

The regression model was significant, \(F(3, 96) = 25.67\), \(p < .001\), \(R^2 = .45\). Both main effects and the interaction effect were statistically significant (see Table 2).

  1. Main Effect of Study Hours: There was a significant main effect of Study Hours on Exam Scores, \(b = 5.30\), \(t(96) = 7.65\), \(p < .001\). This finding suggests that, on average, higher levels of study hours are associated with higher exam scores.

  2. Main Effect of Sleep Hours: The main effect of Sleep Hours on Exam Scores was also significant, \(b = 3.20\), \(t(96) = 4.85\), \(p < .001\), indicating that higher levels of sleep are associated with higher exam scores.

  3. Interaction Effect of Study Hours and Sleep Hours: The interaction between Study Hours and Sleep Hours was significant, \(b = 5.00\), \(t(96) = 6.32\), \(p < .001\). This interaction indicates that the positive relationship between study hours and exam scores is stronger for individuals with higher sleep hours than for those with lower sleep hours (see Figure 1).

Summary and Key Takeaways

  1. Main Effects vs. Interactions:
    • Main effects show isolated impacts, while interactions reveal combined impacts.
  2. Using Means to Detect Interactions:
    • Comparing group means across predictor levels helps detect interactions.
  3. Real-World Relevance:
    • Interactions show how combined factors impact outcomes, which is vital in fields like psychology and public health.
  4. Categorical x Categorical, Linear x Linear, and Linear x Categorical:
    • Use effect coding for categorical predictors in lm() models and factors for visualization.
    • Plotting interactions helps reveal patterns such as cross-over or magnitude effects.
